# 🤖 Sentiment AI Pro - Phân Tích Cảm Xúc Khách Hàng

Ứng dụng phân tích cảm xúc phản hồi từ khách hàng sử dụng AI Studio.

## ✨ Tính năng
- Phân loại: Tích cực, Tiêu cực, Trung lập.
- Hiển thị trực quan với màu sắc và biểu tượng.
- Giải thích lý do từ AI.

## 🛠 Công nghệ
- Frontend: HTML, CSS, JavaScript.
- AI: Google Gemini 1.5 Flash API.
- Đóng gói: Docker.
- Triển khai: Vercel.

## 🚀 Cách chạy Docker
1. `docker build -t sentiment-ai .`
2. `docker run -d -p 8080:80 sentiment-ai`

## ⚠️ Lưu ý
API Key được nhúng trực tiếp trong code cho mục đích chấm bài. Đây là cách làm không an toàn cho môi trường thực tế.
